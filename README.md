# Multiple Choice Indexer

用于生成选择题题库索引的工具，使得开卷考试过程中查找选择题答案的复杂度从O(n)降到O(logn)。

通过正则表达式匹配题号，修改成统一编号的模式，并且通过微软某拼音插件将题库按拼音升序排序构成索引。

## 输入与输出

把选择题题库输入到in.txt中，运行程序，得到out1.txt与out2.txt。
